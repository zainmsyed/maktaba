Maktaba — self-hosted reader. Frontend: frontend/ (SvelteKit). Backend: backend/ (FastAPI) with Postgres + pgvector storing documents, pages, highlights, notes and embeddings. A worker extracts PDF text (PyMuPDF) with optional OCR. Fragile: pgvector/bootstrap, OCR reliability, highlight↔viewer locator sync, and the embedding/job pipeline. Pi helpers live under .pi/.
