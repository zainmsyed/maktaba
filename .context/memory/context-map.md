# Context Map

Compact orientation: this repo contains Pi/Vazir tooling and extensions that support the coding agent. Primary code lives in .pi/extensions (vazir-context, vazir-tracker, vazir-live-reload) and .pi/lib. AGENTS.md holds project intent and onboarding notes. Fragile areas: change-tracking (vazir-tracker), context injection (vazir-context), and live-reload integration. Key paths: AGENTS.md, .context/memory/*.md, and the .pi folder for runtime extensions.
